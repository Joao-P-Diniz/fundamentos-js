Como Criar sua Primeira Página HTML

Aprender HTML é o primeiro passo para quem deseja desenvolver sites. Essa linguagem é responsável por estruturar o conteúdo das páginas da web, permitindo adicionar títulos, textos, imagens, listas e muito mais.

Neste tutorial fictício, veremos uma sequência simples para criar uma página HTML básica utilizando qualquer editor de código, como o Visual Studio Code.

Passos do Tutorial

  1. Crie um arquivo chamado index.html.
  2. Digite a estrutura básica do HTML 5.
  3. Adicione um título utilizando a tag <h1>.
  4. Insira um parágrafo utilizando a tag <p>.
  5. Salve o arquivo e abra-o no navegador.
